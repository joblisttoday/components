Usage

Add favorite buttons to any item with an ID:

<joblist-favorite-button item-id="company-id" item-type="company"></joblist-favorite-button>
<joblist-favorite-button item-id="job-id" item-type="job"></joblist-favorite-button>

Attributes

  • item-id - Unique identifier for the item
  • item-type - Either "company" or "job"

Events

  • favorite-added - Fired when item is added to favorites
  • favorite-removed - Fired when item is removed from favorites
  • storage-required - Fired when storage connection is needed
