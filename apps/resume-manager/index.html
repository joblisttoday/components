Resume Manager

Manage multiple resume variations for different job applications

Resume Manager

Features

  • ✅ Multiple resume versions (default, frontend, backend, etc.)
  • ✅ Easy switching between versions
  • ✅ Create new custom resume variations
  • ✅ Edit/preview functionality
  • ✅ Delete custom versions (default protected)
  • ✅ Plain text format for easy editing
  • ✅ Auto-sync across devices

Use Cases

  • Default - General purpose resume
  • Frontend - Emphasis on React, JavaScript, CSS skills
  • Backend - Focus on APIs, databases, server technologies
  • Fullstack - Balanced frontend and backend experience
  • DevOps - Infrastructure, CI/CD, cloud platforms
  • Manager - Leadership and project management experience

Usage

<joblist-resume-manager></joblist-resume-manager>

How to Use

  1. Connect your storage using the storage widget
  2. Start with the "default" resume
  3. Click "Edit" to modify your resume
  4. Click "+ New" to create variations for different job types
  5. Select different versions from the dropdown
  6. Use "Delete" to remove custom versions (default is protected)

Events

  • resume-saved - Fired when resume is saved
  • resume-deleted - Fired when resume is deleted
