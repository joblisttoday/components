Joblist boards (ATS)

These providers are used to source all jobs of companies registered on joblist.today.

Use this page to prepare a code snippet to embedd a list of job in any web page.

  1. Select a provider-name
  2. Input the provider-hostname for project/company, which can be found in the URL or the provider's job board
  3. Copy and insert the code snippet into any HTML webpage, customize with CSS.

Code snippet & example

Info

provider-name
The name and unique identifier of a job board provider (as in ATS and/or data source)
provider-hostname
The name and unique identifier of a company or project, as given by the job board provider. For example "spacex" is the hostname assigned by the provider "greenhouse".
